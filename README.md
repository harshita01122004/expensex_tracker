# 📱 ExpenseX Tracker

## 📌 Overview

ExpenseX Tracker is an Android-based personal expense management application developed using Kotlin and Android Studio. The application helps users efficiently track, manage, and analyze their daily expenses through a simple and user-friendly interface.

---

## 🚀 Features

* 🔐 User Authentication (Login & Registration)
* ➕ Add and manage daily expenses
* 📊 Categorize expenses (Food, Travel, Shopping, etc.)
* 📅 Store details like title, amount, date, and category
* 📈 Automatic total expense calculation
* 📋 Dynamic expense list using RecyclerView
* 🎨 Clean UI based on Material Design
* 📱 Fully offline application (no internet required)

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **IDE:** Android Studio
* **Frontend:** XML Layouts
* **Backend Logic:** Kotlin (Android Framework)
* **Database:** SQLite
* **UI Components:** RecyclerView, CardView, Floating Action Button

---

## 🏗️ System Architecture

The application follows a modular architecture:

* **View:** User Interface (XML)
* **Controller:** Application Logic (Kotlin)
* **Model:** SQLite Database
